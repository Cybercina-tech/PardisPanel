# Landing Page Images Directory

Place your landing page images here:

## Required Images

1. **demo-screenshot.png** (or .jpg)
   - Dashboard/preview screenshot
   - Recommended size: 1920x1080px or similar
   - Format: PNG or JPG
   - Already referenced in `index.html`

## Optional Images

2. **logo.png** (or .svg)
   - Your company logo
   - Recommended: SVG for scalability, or PNG with transparent background
   - Update navbar in `index.html` to use this

3. **clients/** (directory)
   - Client/exchange logos for "Trusted by" section
   - Recommended: 200x100px each
   - Update trusted logos section in `index.html`

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Keep file sizes under 500KB for fast loading
- Use WebP format for better compression (with fallbacks)
- Consider lazy loading for images below the fold
